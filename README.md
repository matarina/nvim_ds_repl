# nvim_ds_repl
a!!!!!!!1
nvim_ds_repl is a quite simple newly created custom nvim plugin for repl. The plugin support R and Python languages.
Most code sending action are inspired Rstudio and Jupyter.
Features:
Sending semantic code block utilize Treesitter object.
Visual selections sending support.
send whole buffer.
Usage


the plugin are based nvim-python-repl
